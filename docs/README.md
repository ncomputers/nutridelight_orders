## Key Concepts

### System Architecture
- **Frontend**: React + TypeScript with modern UI components
- **Backend**: Supabase (PostgreSQL + real-time API)
- **Architecture**: Clean layered architecture with feature-based organization

### User Roles
- **Restaurants**: Place daily produce orders
- **Admin Staff**: Manage orders and system operations
- **Purchase Team**: Plan and execute stock purchases
- **Sales Team**: Handle invoicing and collections

### Core Workflows
- **Order Processing**: From placement to delivery
- **Purchase Planning**: Demand aggregation and procurement
- **Sales Invoicing**: Billing and payment collection
- **Accounting**: Double-entry bookkeeping system
