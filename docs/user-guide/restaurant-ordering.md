# Restaurant Ordering Guide

## Overview

This guide helps restaurant staff place daily produce orders through the Nutridelight Orders web application.

## Getting Started

### Access Requirements
- Web browser (Chrome, Firefox, Safari, Edge)
- Restaurant-specific order URL (provided by Nutridelight)
- No login required for ordering

### Order URL Format
```
https://app.nutridelight.com/order?r=[restaurant-slug]
```

Your restaurant slug will be provided by the Nutridelight team.

## Ordering Process

### Step 1: Open Order Page
1. Navigate to your restaurant's order URL
2. The page will display available produce items
3. Items are organized by categories: Vegetables, Herbs & Greens, Premium Fruits

### Step 2: Select Items
1. Browse items by category
2. Each item shows:
   - English and Hindi names
   - Current availability status
   - Quantity input field
3. Enter desired quantities in kilograms (kg)
4. Use increment/decrement buttons for quick adjustments

### Step 3: Review Order
1. Selected items appear in the order summary
2. Review quantities and make adjustments
3. Add special notes if needed (max 300 characters)
4. Total selected items count is displayed

### Step 4: Submit Order
1. Click "Place Order" button
2. Confirm order details in the popup
3. Receive order confirmation with reference number
4. Save the order reference for tracking

## Order Features

### Quantity Management
- **Increase**: Click + button or type higher value
- **Decrease**: Click - button or type lower value
- **Precision**: Quantities rounded to 0.1 kg
- **Minimum**: 0 kg (removes item from order)

### Item Availability
- **Available**: Item can be ordered
- **Out of Stock**: Item temporarily unavailable
- **Limited**: Restricted quantities available

### Order Timing
- **Order Deadline**: 11:00 PM daily
- **Delivery**: Next-day delivery
- **Confirmation**: By 7:00 AM on delivery day

## Order Status Tracking

### Status Flow
1. **Pending** - Order received, awaiting confirmation
2. **Confirmed** - Order verified and scheduled
3. **Purchase Done** - Items procured from suppliers
4. **Out for Delivery** - Order on the way
5. **Delivered** - Order received at restaurant

### Checking Status
- Use order reference number
- Contact Nutridelight support
- Wait for confirmation calls/messages

## Important Notes

### Order Confirmation
- Orders are requests, not confirmed purchases
- Final confirmation depends on:
  - Market availability
  - Daily mandi (market) rates
  - Supply constraints

### Payment
- No payment collected during ordering
- Payment arrangements handled separately
- Invoices generated after delivery

### Communication
- Confirmation call by 7:00 AM
- Updates on availability changes
- Delivery time notifications

## Troubleshooting

### Common Issues

**Page Not Loading**
- Check internet connection
- Verify URL is correct
- Try different browser
- Contact support if persists

**Items Not Available**
- Items marked "Out of Stock" cannot be ordered
- Check back later for availability updates
- Contact support for special requests

**Quantity Errors**
- Ensure valid numeric values
- Check minimum order quantities
- Use decimal point for partial kilograms

**Order Submission Failed**
- Check all required fields are filled
- Verify internet connection
- Try refreshing the page
- Contact support if issue persists
